## Getting Started

### Prerequisites

### Installation

1.  Clone the repo

```sh
git clone https://github.com/p-bizouard/ShareMyRoster.git
cd ShareMyRoster
```

2.  Install NPM packages

```sh
yarn install
```

3.  Run dev server

```sh
yarn start
```
